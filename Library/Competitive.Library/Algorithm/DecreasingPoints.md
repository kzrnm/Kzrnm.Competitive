## 概要

点の集合から単調減少となるように点を選び取る。

```

Y
↑
│ +
│
│       +
│
│        +  +
│    +           +  +
│           +  +
│                     +
└───────────────────────→ X
```

上記のような点の集合があるとする。

```
Y
↑
│ *
│
│       *
│
│        $  *
│    +           $  *
│           +  +
│                     *
└───────────────────────→ X
```

`*` は採用される。`$` は strict じゃないときは採用される。
